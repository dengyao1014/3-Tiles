# 3 Tiles 游戏网站

## 项目描述
3 Tiles 是一个基于网页的拼图游戏，提供沉浸式的游戏体验和丰富的游戏攻略。

## 功能特点
- 响应式网页设计
- 清晰的游戏说明
- 详细的博客文章和游戏攻略
- 现代化的用户界面

## 技术栈
- HTML5
- CSS3
- JavaScript

## 项目结构
- `index.html`: 主页
- `blog.html`: 博客列表页
- `styles.css`: 全局样式表
- `blog/`: 博客文章目录

## 如何运行
直接在浏览器中打开 `index.html` 即可
